# LOT SIZE CALCULATOR 2

A professional risk management suite for traders — Position Sizing, Pip Value, P&L, Risk:Reward, Margin, Prop Firm Rules, Drawdown Simulator, Trade Journal, and Risk Dashboard.

## Features

- **Position Sizing Calculator** — Compute lot size based on account balance, risk %, entry/stop/take-profit levels, spread, and commission
- **Pip Value Calculator** — Calculate pip, tick, and point values across any asset
- **Profit / P&L Calculator** — Profit/loss, return %, and pip distance
- **Risk : Reward Calculator** — R:R ratio with break-even win rate and trade quality
- **Margin Calculator** — Margin required, position value, effective leverage, account usage %
- **Prop Firm Rules Checker** — FTMO, MyForexFunds, FundedNext, The5%ers, Goat Funded presets
- **Drawdown Simulator** — Simulate consecutive losses at a fixed risk %
- **Trade Journal** — Log, track, export/import trades with win rate and profit factor stats
- **Risk Dashboard** — Live risk limits, heatmap, and multi-position risk aggregation

## Tech Stack

Vanilla HTML, CSS, JavaScript — no dependencies required.

## Usage

Open `index.html` in any modern browser.
